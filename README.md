# Efficiency-Models-with-Machine-Learning
This repository contains the code developed for Assignment 2 of the Advanced Statistical Machine Learning (MXN441) course. The assignment is a required component for the successful completion of the course.
It contains a replication of the methodology proposed in the paper "Integrating Relative Efficiency Models with Machine Learning Algorithms for Performance Prediction" (2024), followed by an extension of the original methodology through the inclusion of an additional variable and the implementation of additional machine learning algorithms. The objective is to validate the original findings and explore potential improvements in predictive performance.

Paper link:

https://journals.sagepub.com/doi/full/10.1177/21582440241257800
